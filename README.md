# What-Bytes Assignment

## Overview
This project is built using modern web technologies to deliver a seamless user experience.

## Tech Stack
- **Frontend:** Next.js, TypeScript, ShadCN UI, Tailwind CSS
- **Charts:** Recharts
- **Icons:** Lucide-react
